Football Scoreboard App

A simple interactive scoreboard to track points for two teams. Built to practice JavaScript DOM manipulation and dynamic interactivity.

Features
- Add points to each team
- Reset/clear scores
- Real-time updates on the page
- Interactive buttons and basic styling

Built With
- HTML5
- CSS3
- JavaScript

How to Use
1. Click '+1', '+2', '+3' button to add respective points to each team
2. Click 'Reset' to clear the scores
